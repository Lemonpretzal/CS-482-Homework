The 17 images below should cover the requirements for Homework #1. I began by reading in the initial image; as you can see I chose grayscale for this assignment. Next I resized that image to 512x512 pixels using cv2.resize().

For the next 3 images, I used cv2.GaussianBlur(), setting sigma to 1, 2, and 3 respectively. As professor Duric pointed out in class, the kernel size has a minimum value based on the value of sigma, therefore the kernel sizes for these images are as follows: 7x7, 13x13, and 19x19 respectively.

Next, I found the derivatives in the x and y directions for each of the previous 4 images (resized image, blurred image 1, blurred image 2, blurred image 3) using cv2.Sobel() and have them displayed below. Each instance of cv2.Sobel() uses a kernel size of 5x5, and output datatype of cv2.CV_64F.

The next 4 images are the magnitudes of the original 4 images. To do this, I used cv2.Canny(). I understand that cv2.Canny() is really edge detection, rather than an actual magnitude. I read through the documentation, and it does actually perform the same magnitude operation, however the image is based on a range, where a pixel is white if the magnitude is greater than a minimum and less than a maximum. The documentation and tutorial I watched set the minimum to 100 and the maximum to 200, so I did the same.

As for applying filters to each color channel separately, Professor Duric told me not to worry about it, since I am using grayscale.
